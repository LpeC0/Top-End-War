\# Top End War — Meta Loop + Upgrade Economy v1

\_Why the player comes back\_



\---



\## 0. Belgenin amacı



Bu belge, World 1 için run dışı ilerleme döngüsünü tanımlar.



Şunları kilitler:

\- oyuncu run bitince ne kazanır

\- neyi yükseltir

\- neyi kalıcı açar

\- hangi kararları verir

\- neden tekrar run başlatmak ister



\### Temel ilke

Meta loop:

\- saf stat şişirmesi olmamalı

\- oyuncuyu grind mezarlığına sokmamalı

\- build denemeye teşvik etmeli

\- World 1 boyunca anlamlı ama kontrollü büyüme vermeli
